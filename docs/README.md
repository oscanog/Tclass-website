# TClass Docs Index

Documentation for the TClass frontend repo and related local setup workflow.

## Start Here

- `../LOCAL_SETUP.md` - full local setup guide (frontend + backend)
- `./WORKSTATION_SETUP_CHECKLIST.md` - new device checklist, installs, environment setup, and reusable project handoff summary
- `./AI_PROMPT.md` - current project context prompt for AI sessions
- `./frontend-setup.md` - frontend-only setup guide

## Repositories

- Frontend: `https://github.com/primex-joseph/Tclass-website.git`
- Backend: `https://github.com/primex-joseph/Tclass-website-backend.git`

## Local Dev URLs (default)

- Frontend: `http://localhost:3000`
- Backend: `http://127.0.0.1:8000`
- Backend API: `http://127.0.0.1:8000/api`

## Notes

- Current frontend stack in this repo is Next.js 16 + React 19 + Tailwind CSS 4.
- Backend docs and setup live in the backend repo (`tclass-v1-backend/docs/`).
- If you clone to a new machine, use `docs/WORKSTATION_SETUP_CHECKLIST.md` first.
