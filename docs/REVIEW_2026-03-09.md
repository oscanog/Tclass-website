## Findings (Priority Order)

1. Build-blocking TypeScript error in admin PDF export
   - File: `app/(faculty-admin)/admin/page.tsx` (around line 1730)
   - Symptom: `BlobPart` typing mismatch for `pdfBytes` from `pdf.save()`
   - Risk: breaks strict TypeScript check and CI gates that require `tsc --noEmit`

2. Build-blocking TypeScript error in student report PDF export
   - File: `app/student/_components/student-sections.tsx` (around line 710)
   - Symptom: same `BlobPart` typing mismatch for `pdfBytes`
   - Risk: same as above

## Recommended Fix (Both Files)
- Convert to concrete `Uint8Array` before creating `Blob`, for example:
  - `const safeBytes = pdfBytes instanceof Uint8Array ? pdfBytes : new Uint8Array(pdfBytes);`
  - `const blob = new Blob([safeBytes], { type: "application/pdf" });`
